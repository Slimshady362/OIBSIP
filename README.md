# OIBSIP Cybersecurity Internship

This repository contains the tasks completed as part of the Oasis Infobyte (OIBSIP) Cybersecurity Internship.

## Internship Objective

The internship focuses on developing practical cybersecurity skills through hands-on projects and research-based assignments. Each task is documented with relevant reports, scan results, screenshots, and supporting files.
